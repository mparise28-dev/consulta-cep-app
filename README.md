# Consulta de CEP

Aplicação web desenvolvida com Node.js e Express para consulta de endereços a partir de um CEP, utilizando a API pública ViaCEP.

## 🚀 Funcionalidades

- Consulta de CEP em tempo real
- Validação de entrada (8 dígitos)
- Exibição dinâmica dos dados
- Tratamento de erros
- Interface simples e responsiva

## 🛠️ Tecnologias utilizadas

- Node.js
- Express
- JavaScript (frontend)
- HTML e CSS

## 📦 Como executar o projeto

1. Clone o repositório:
git clone https://github.com/mparise28-dev/consulta-cep-app.git

2. Instale as dependências:
npm install

3. Inicie o servidor:
node server.js

4. Acesse no navegador:
http://localhost:3000

## 🌐 API utilizada

ViaCEP:
https://viacep.com.br/

## 📌 Observações

Este projeto foi desenvolvido como atividade acadêmica para prática de requisições HTTP e integração entre frontend e backend.
